Banco - Back End com Pyhton-Django

Back End do projeto integrador do segundo semestre do curso de Desenvolvimento de sistemas da instituição SENAI. O projeto foi criado utilizando a biblioteca de criação de aplicações web Django com REST Framework para a criação de uma API REST.

A api tinha como principais funcionalidades:

  • Criação de usuário (comum e administrador)
  • Solicitação de cartão de crédito
  • Solicitação de empréstimo
  • Transferencias bancárias

Banco - Mobile com React Native

Aplicação mobile do projeto integrador do segundo semestre do curso de Desenvolvimento de sistemas da instituição SENAI. A parte mobile do porjeto foi criado utilizando framework React Native. Toda a usabilidade do APP foi feita coma integração da API.

O app incluia as seguintes funcionalidades

  • Criação de conta
  • Login
  • Solicitação de cartão de crédito
  • Solicitação de empréstimo
  • Transferencias bancárias

Banco - Website com ReactJS

Website do projeto integrador do segundo semestre do curso de Desenvolvimento de sistemas da instituição SENAI. Landing page com próposito de apresentar o produto ao cliente. O site permitia o cliente de acessar sua conta e verificar o saldo.

Descubra o número - Java

Jogo de descobrir um número randomico gerado pelo computador utilizando o paradigma de Orientação a Objeto e a biblioteca Swing para criação da interface do jogo.
